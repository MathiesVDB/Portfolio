AI Agent

This was a project I did for the Gameplay Programming course at Howest DAE. Together with the research project.
The goal was to write an AI agent that could make simple decisions in a zombie survival game in order to survive.

The agent made decisions based on a behaviour tree.


The agent was also able to walk around the world along a grid to make sure he did not walk through the same area multiple times.


The agent also managed its own inventory, by finding items in the world and picking them up.
Then my memory class would remember the items and which inventory slot they were in.
All that was left after that was to let the behaviour tree decide which item to use at which time.
